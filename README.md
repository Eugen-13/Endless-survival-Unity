# EndlessSurvival

Простая игра на Unity в жанре выживания.

Враги идут к игроку через NavMesh. После убийства выпадают кристаллы. За кристаллы можно улучшать статы персонажа (здоровье, урон, скорость и т.д.). Прогресс сохраняет
